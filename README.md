# Interactive-Shopping-Cart-System-using-Python

##  Project Overview
This project is a simple shopping cart billing system developed using Python. It allows users to enter item prices continuously, applies discounts based on the total amount, and generates a final bill summary.

---

##  Objectives
- Accept multiple item prices using a loop
- Validate user input (no negative values allowed)
- Calculate subtotal and item count
- Apply discount rules based on total amount
- Display a clear and structured bill summary

---

##  Features
- Sentinel-controlled input (enter 0 to stop)
- Input validation for negative values
- Automatic discount calculation:
  - 20% for totals ≥ 1000
  - 10% for totals ≥ 500
  - 5% for totals ≥ 200
  - No discount for totals < 200
- Clean output formatting
